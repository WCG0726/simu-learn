## 项目简介

本平台是一个综合仿真学习平台，涵盖 COMSOL、ANSYS、DEFORM 等软件的教程，涉及电子封装、热电材料、金属成形、热-电-力耦合等领域的仿真。

## 功能特点

- 📚 **入门教程** - 软件安装、界面介绍、基础操作
- 🔬 **应用案例** - 电子封装、热电材料、金属成形
- 🛠️ **实用工具** - 参数计算器、材料数据库
- ❓ **常见问题** - 问题诊断和解决方案

## 支持的仿真软件

| 软件 | 应用领域 | 教程链接 |
|------|----------|----------|
| **COMSOL** | 多物理场耦合 | [COMSOL 教程](/comsol/) |
| **ANSYS** | 结构、流体、电磁 | [ANSYS 教程](/ansys/) |
| **DEFORM** | 金属成形 | [DEFORM 教程](/deform/) |

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建静态文件

```bash
npm run build
```

### 部署到 GitHub Pages

```bash
npm run deploy
```

## 技术栈

- **VitePress** - 静态站点生成器
- **Markdown** - 内容编写
- **GitHub Pages** - 部署托管
