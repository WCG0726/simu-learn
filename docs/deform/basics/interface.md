# DEFORM 界面介绍

本节介绍 DEFORM 的主要界面元素和基本操作。

## 🖥️ 主界面布局

```
┌─────────────────────────────────────────────────────────┐
│  菜单栏  File  Edit  View  Pre-Processor  Simulator  ...│
├─────────────────────────────────────────────────────────┤
│  工具栏  [新建] [打开] [保存] [撤销] [重做] [运行]        │
├──────────┬──────────────────────────────┬───────────────┤
│          │                              │               │
│  项目树   │        图形窗口              │   属性面板    │
│          │                              │               │
│  📁 项目  │      ┌──────────┐           │   对象:       │
│    ├ 几何 │      │          │           │   类型:       │
│    ├ 材料 │      │  工件/模具│           │   材料:       │
│    ├ 接触 │      │          │           │   温度:       │
│    ├ 边界 │      └──────────┘           │               │
│    └ 步骤 │                              │               │
│          │                              │               │
├──────────┴──────────────────────────────┴───────────────┤

```

## 📋 主要组件

### 1. 菜单栏

| 菜单 | 功能 | 说明 |
|------|------|------|
| **File** | 文件操作 | 新建、打开、保存、导入/导出 |
| **Edit** | 编辑操作 | 撤销、重做、复制、粘贴 |
| **View** | 视图控制 | 显示/隐藏、缩放、旋转 |
| **Pre-Processor** | 前处理 | 几何、材料、边界条件设置 |
| **Simulator** | 求解器 | 运行计算、监控进度 |
| **Post-Processor** | 后处理 | 结果查看、数据提取 |
| **Tools** | 工具 | 选项、宏、脚本 |
| **Help** | 帮助 | 文档、关于 |

### 3. 项目树

项目树显示仿真项目的层次结构：

```
📁 Simulation Project
├── 📦 Geometry
│   ├── Workpiece (工件)
│   └── Die (模具)
├── 🧱 Material
│   ├── Workpiece Material
│   └── Die Material
├── 🤝 Contact
│   └── Workpiece-Die Contact
├── 🎯 Boundary Conditions
│   ├── Velocity/Force
│   └── Temperature
├── 🔄 Process Steps
│   ├── Step 1: Forging
│   └── Step 2: Cooling
└── ⚙️ Simulation Control
    ├── Time Step
    └── Solver Settings
```

### 4. 图形窗口

图形窗口显示几何模型和仿真结果：

**视图控制：**
- **旋转**：鼠标左键拖动
- **缩放**：鼠标滚轮
- **平移**：鼠标中键拖动
- **适合窗口**：双击中键

**显示模式：**
- **线框模式**：Wireframe
- **实体模式**：Solid
- **网格模式**：Mesh
- **结果模式**：Results

## 🎛️ 前处理器界面

### 几何导入

```
Pre-Processor → Geometry → Import
├── 文件类型: STL, IGES, STEP
├── 单位: mm, cm, m
└── 位置: 坐标原点或指定位置
```

### 材料设置

```
Pre-Processor → Material
├── 从库中选择
│   ├── Steel (钢)
│   ├── Aluminum (铝)
│   ├── Copper (铜)
│   └── Titanium (钛)
└── 自定义材料
    ├── 弹性模量
    ├── 泊松比
    ├── 流变应力曲线
    └── 热物性参数
```

### 接触设置

```
Pre-Processor → Contact
├── 接触类型
│   ├── Master-Slave
│   └── 自动检测
├── 摩擦模型
│   ├── Coulomb (库仑)
│   ├── Shear (剪切)
│   └── None (无摩擦)
└── 摩擦系数
    └── μ = 0.3 (典型值)
```

### 边界条件

```
Pre-Processor → Boundary Conditions
├── 速度边界
│   ├── 模具速度
│   └── 工件约束
├── 力边界
│   └── 载荷大小
├── 温度边界
│   ├── 初始温度
│   └── 对流换热
└── 对称边界
    └── 2D 或 3D 对称
```

## ⚙️ 求解器界面

### 运行仿真

```
Simulator → Run
├── 步骤数: 100
├── 时间步: 自动或手动
├── 保存间隔: 每 10 步
└── 启动计算
```

### 监控进度

```
Simulator → Monitor
├── 进度条: 50%
├── 当前步: 50/100
├── 计算时间: 00:05:23
├── 预计剩余: 00:05:23
└── 实时曲线
    ├── 载荷-位移
    └── 温度-时间
```

## 📊 后处理器界面

### 结果查看

```
Post-Processor → Results
├── 应变分布
│   ├── 等效应变
│   └── 主应变
├── 应力分布
│   ├── 等效应力
│   └── 主应力
├── 温度分布
│   └── 温度场
├── 速度分布
│   └── 速度场
└── 损伤分布
    └── 损伤因子
```

### 数据提取

```
Post-Processor → Data Extraction
├── 点数据
│   └── 特定点的变量值
├── 线数据
│   └── 沿路径的变量分布
├── 面数据
│   └── 截面上的变量分布
└── 体数据
    └── 整体统计量
```

### 动画制作

```
Post-Processor → Animation
├── 帧范围: 0-100
├── 帧率: 30 fps
├── 输出格式: AVI, MP4, GIF
└── 导出动画
```

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+N | 新建项目 |
| Ctrl+O | 打开项目 |
| Ctrl+S | 保存项目 |
| Ctrl+Z | 撤销 |
| Ctrl+Y | 重做 |
| F5 | 刷新 |
| F7 | 运行仿真 |
| F8 | 暂停仿真 |
| F9 | 停止仿真 |
