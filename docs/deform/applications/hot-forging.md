# 热锻仿真

热锻是 DEFORM 最经典的应用场景。本节介绍如何使用 DEFORM 进行热锻过程仿真。

## 📚 热锻基础

### 什么是热锻？

热锻是在再结晶温度以上进行的塑性成形工艺。由于高温下金属的流动应力降低，可以获得较大的变形量和良好的成形性。

### 热锻温度范围

| 材料 | 始锻温度 (°C) | 终锻温度 (°C) |
|------|---------------|---------------|
| 碳钢 | 1200-1250 | 800-850 |
| 合金钢 | 1150-1200 | 850-900 |
| 铝合金 | 400-500 | 350-400 |
| 钛合金 | 900-950 | 800-850 |
| 镍基合金 | 1100-1150 | 900-950 |

## 🔧 仿真设置

### 步骤 1：创建几何

1. **导入 CAD 模型**
   ```
   Pre-Processor → Geometry → Import
   ├── 文件类型: STL
   ├── 单位: mm
   └── 导入工件和模具
   ```

2. **或使用 DEFORM 建模**
   ```
   Pre-Processor → Geometry → Create
   ├── 圆柱体
   ├── 长方体
   └── 自定义形状
   ```

### 步骤 2：定义材料

1. **从材料库选择**
   ```
   Pre-Processor → Material → Library
   ├── 选择材料: AISI-1045
   └── 添加到工件
   ```

2. **材料参数**
   ```
   材料属性:
   ├── 密度: 7850 kg/m³
   ├── 弹性模量: 210 GPa
   ├── 泊松比: 0.3
   ├── 热导率: 45 W/(m·K)
   ├── 比热容: 480 J/(kg·K)
   └── 流变应力: σ = f(ε, ε̇, T)
   ```

### 步骤 3：设置边界条件

1. **温度边界**
   ```
   边界条件 → 温度
   ├── 工件初始温度: 1100 °C
   ├── 模具初始温度: 200 °C
   └── 环境温度: 25 °C
   ```

2. **速度边界**
   ```
   边界条件 → 速度
   ├── 上模速度: 100 mm/s
   ├── 下模固定
   └── 工件对称约束
   ```

3. **换热边界**
   ```
   边界条件 → 换热
   ├── 工件-模具: h = 5000 W/(m²·K)
   ├── 工件-空气: h = 20 W/(m²·K)
   └── 模具-空气: h = 20 W/(m²·K)
   ```

### 步骤 4：设置接触

```
接触设置:
├── 接触对: 工件-模具
├── 摩擦模型: Shear
├── 摩擦系数: 0.3
└── 热传导系数: 5000 W/(m²·K)
```

### 步骤 5：设置仿真参数

```
仿真控制:
├── 步骤数: 100
├── 时间步: 自动 (0.001s)
├── 保存间隔: 每 10 步
└── 总行程: 50 mm
```

## 📊 结果分析

### 应变分布

查看等效应变分布：
1. Post-Processor → Strain → Effective Strain
2. 观察应变集中区域
3. 评估变形均匀性

### 应力分布

查看等效应力分布：
1. Post-Processor → Stress → Effective Stress
2. 识别应力集中位置
3. 评估模具载荷

### 温度分布

查看温度场变化：
1. Post-Processor → Temperature
2. 观察温度下降趋势
3. 评估成形窗口

### 载荷曲线

查看载荷-位移曲线：
1. Post-Processor → Graphs → Load vs Stroke
2. 分析载荷变化
3. 预测设备吨位

## 💡 优化建议

### 减少载荷

| 措施 | 效果 |
|------|------|
| 提高始锻温度 | 显著降低载荷 |
| 降低摩擦系数 | 中等降低载荷 |
| 优化预锻形状 | 减少变形量 |
| 增加工步 | 分散载荷 |

### 提高模具寿命

| 措施 | 效果 |
|------|------|
| 降低模具温度 | 减少热疲劳 |
| 优化模具圆角 | 减少应力集中 |
| 使用润滑 | 降低摩擦 |
| 预热模具 | 减少热冲击 |

### 避免缺陷

| 缺陷 | 原因 | 解决方案 |
|------|------|----------|
| 折叠 | 金属流动不合理 | 优化模具设计 |
| 充不满 | 载荷不足 | 增加行程或预锻 |
| 过热 | 温度过高 | 降低始锻温度 |
| 开裂 | 变形过大 | 增加工步 |
