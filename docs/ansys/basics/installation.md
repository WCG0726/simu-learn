# ANSYS 安装与配置

本教程将指导你完成 ANSYS 的安装和基本配置。

## 📋 系统要求

### 硬件要求
| 配置 | 最低要求 | 推荐配置 |
|------|----------|----------|
| **处理器** | Intel/AMD 64位 | 多核处理器 (16核+) |
| **内存** | 8 GB | 64 GB 或更高 |
| **硬盘** | 20 GB | SSD 200 GB+ |
| **显卡** | 支持 OpenGL | 专业显卡 (NVIDIA Quadro/RTX) |

### 操作系统
- Windows 10/11 (64位) ✅ 推荐
- Linux (RHEL 7/8, CentOS 7/8, Ubuntu 18.04/20.04)

## 🔧 安装步骤

### 步骤 1：下载安装包
1. 访问 [ANSYS 官网](https://www.ansys.com)
2. 登录或注册账号
3. 下载 ANSYS 安装管理器

### 步骤 2：运行安装程序

**Windows:**
```
双击运行 setup.exe
→ 选择"Install ANSYS Products"
→ 接受许可协议
→ 选择安装目录
```

### 步骤 3：选择产品

推荐安装的产品：

| 产品 | 用途 | 推荐 |
|------|------|------|
| **Workbench** | 统一仿真平台 | ✅ 必装 |
| **Mechanical APDL** | 结构分析 | ✅ 推荐 |
| **Mechanical** | 结构分析（GUI） | ✅ 推荐 |
| **Fluent** | 流体分析 | ✅ 推荐 |
| **CFX** | 流体分析 | 可选 |
| **Icepak** | 电子热管理 | ✅ 电子封装推荐 |
| **Maxwell** | 电磁分析 | 可选 |
| **SpaceClaim** | 几何建模 | ✅ 推荐 |

### 步骤 4：配置许可证

**学生版（免费）：**
- 下载 ANSYS Student 版
- 功能有限但免费

**商业版：**
```
许可证配置:
├── 许可证类型: FlexNet
├── 服务器地址: port@server
└── 例如: 2325@license-server.company.com
```

## 📊 典型工作流程

### 步骤 1：创建项目

1. 从工具箱拖动分析系统到项目原理图
2. 或双击分析系统自动创建

### 步骤 2：定义工程数据

```
双击 "Engineering Data"
├── 添加材料
│   ├── 从库中选择
│   └── 或创建新材料
└── 定义材料属性
    ├── 密度
    ├── 弹性模量
    ├── 泊松比
    └── 热导率等
```

### 步骤 3：创建几何

```
双击 "Geometry"
├── 打开 DesignModeler 或 SpaceClaim
├── 创建几何
└── 保存并关闭
```

### 步骤 4：模型设置

```
双击 "Model"
├── 材料分配
├── 接触设置
├── 网格划分
└── 网格质量检查
```

### 步骤 5：设置边界条件

```
双击 "Setup"
├── 边界条件
│   ├── 固定约束
│   ├── 载荷
│   └── 温度
└── 分析设置
```

### 步骤 6：求解

```
右键 "Solution" → Solve
├── 检查求解状态
└── 查看警告/错误
```

### 步骤 7：后处理

```
双击 "Results"
├── 添加结果
│   ├── 变形
│   ├── 应力
│   └── 应变
├── 生成图表
└── 导出报告
```

## 🔧 常用设置

### 并行计算设置

```
Tools → Options → Analysis Settings and Solution
├── 并行求解器: 启用
├── CPU 数量: 根据系统设置
└── GPU 加速: 如果有专业显卡
```

### 内存设置

```
Tools → Options → Memory
├── 最大内存使用: 80%
└── 临时文件目录: SSD 路径
```

### 单位设置

```
Tools → Units
├── 长度: mm
├── 力: N
├── 时间: s
├── 温度: °C
└── 或选择预设: Metric (mm, kg, s)
```

## 🚨 常见问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 启动失败 | 许可证问题 | 检查许可证服务器连接 |
| 内存不足 | 模型太大 | 增加内存或简化模型 |
| 网格失败 | 几何问题 | 修复几何或调整网格设置 |
| 不收敛 | 边界条件问题 | 调整载荷步或边界条件 |
| 显示异常 | 显卡驱动 | 更新显卡驱动 |
