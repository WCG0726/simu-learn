## 📐 常用公式

### 热阻计算
```
Rth = ΔT / P = L / (k × A)
```
- Rth: 热阻 (K/W)
- ΔT: 温差 (K)
- P: 功率 (W)
- L: 长度 (m)
- k: 热导率 (W/(m·K))
- A: 面积 (m²)

### 焦耳热
```
Q = I² × R = J² / σ × V
```
- Q: 热功率 (W)
- I: 电流 (A)
- R: 电阻 (Ω)
- J: 电流密度 (A/m²)
- σ: 电导率 (S/m)
- V: 体积 (m³)

### 热应力
```
σ = E × α × ΔT
```
- σ: 应力 (Pa)
- E: 杨氏模量 (Pa)
- α: 热膨胀系数 (1/K)
- ΔT: 温差 (K)
