## 📖 使用建议

### 材料选择原则

| 应用场景 | 关键参数 | 推荐材料 |
|----------|----------|----------|
| 高散热 | 热导率 | Cu, Al, AlN |
| 低热膨胀 | CTE | W, Mo, Invar |
| 电绝缘 | 电阻率 | Al₂O₃, FR-4 |
| 高温应用 | 热稳定性 | SiC, AlN |
| 低成本 | 价格 | Al, FR-4 |

### 参数单位换算

```
热导率: 1 W/(m·K) = 0.01 W/(cm·K) = 0.239 cal/(cm·s·°C)
电阻率: 1 Ω·m = 100 Ω·cm = 10⁶ Ω·μm
CTE: 1 ppm/K = 10⁻⁶ /K
```
