# COMSOL 界面介绍

本节介绍 COMSOL Multiphysics 的主要界面元素和基本操作。

## 📋 主要组件

### 1. 模型树（Model Tree）
模型树位于左侧，显示仿真的完整结构：

- **组件** - 几何、物理场、网格
- **研究** - 求解设置、结果
- **结果** - 图表、数据导出

### 2. 图形窗口（Graphics Window）
中央区域显示几何模型和结果：

- **视图控制** - 旋转、缩放、平移
- **选择工具** - 点、边、面、体选择
- **显示选项** - 网格、结果、透明度

### 3. 设置面板（Settings Panel）
右侧显示当前选中项的详细设置：

- **参数输入** - 数值、表达式
- **下拉选择** - 选项列表
- **复选框** - 开关选项

## 🔧 常用操作

### 选择操作
- **单击** - 选择单个对象
- **Ctrl+单击** - 添加/取消选择
- **框选** - 选择区域内所有对象
- **全选** - Ctrl+A

### 视图控制
- **旋转**：鼠标左键拖动
- **缩放**：鼠标滚轮
- **平移**：鼠标中键拖动
- **适合窗口**：Ctrl+F

### 快捷键
| 快捷键 | 功能 |
|--------|------|
| Ctrl+S | 保存 |
| Ctrl+Z | 撤销 |
| Ctrl+Y | 重做 |
| F5 | 刷新视图 |
| F7 | 计算 |

## 📊 结果可视化

### 切面图
1. 在结果 → 2D 切面图中创建
2. 选择显示的数据
3. 调整颜色范围

### 等值面
1. 在结果 → 3D 绘图组中创建
2. 添加等值面
3. 设置等值面值

### 动画
1. 右键点击绘图组
2. 选择"添加动画"
3. 设置时间范围和帧数
