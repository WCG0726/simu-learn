# 几何建模

本节介绍 COMSOL 中的几何建模功能。

## 📐 几何建模基础

COMSOL 提供了强大的几何建模工具，支持 2D 和 3D 几何创建。

### 2D 几何工具
- **矩形** - 创建矩形区域
- **圆/椭圆** - 创建圆形或椭圆形区域
- **多边形** - 创建任意多边形
- **贝塞尔曲线** - 创建曲线边界

### 3D 几何工具
- **长方体** - 创建长方体
- **球体** - 创建球体
- **圆柱体** - 创建圆柱体
- **圆锥体** - 创建圆锥体
- **拉伸** - 将 2D 几何拉伸为 3D
- **旋转** - 将 2D 几何旋转为 3D

## 🔧 布尔运算

| 操作 | 说明 |
|------|------|
| 并集 | 合并多个几何体 |
| 交集 | 保留重叠部分 |
| 差集 | 从一个几何体中减去另一个 |

## 💡 建模技巧

1. **参数化设计** - 使用参数定义尺寸，便于修改
2. **利用对称性** - 只建模一部分，减少计算量
3. **简化细节** - 去掉对结果影响不大的小特征

---

::: tip 提示
建议先在草图软件中规划好几何结构，再在 COMSOL 中建模。
:::
