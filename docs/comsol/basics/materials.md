# 材料设置

本节介绍如何在 COMSOL 中设置材料属性。

## 📋 材料库

COMSOL 内置了丰富的材料库，包含常用材料的属性数据。

### 访问材料库
1. 在模型树中右键点击 **材料**
2. 选择 **从库添加材料**
3. 搜索或浏览所需材料

### 材料属性分类
| 类型 | 属性 |
|------|------|
| 热学 | 热导率、比热容、密度 |
| 电学 | 电导率、介电常数 |
| 力学 | 杨氏模量、泊松比 |

## 🔧 自定义材料

1. 右键点击 **材料** → **添加材料**
2. 输入材料名称
3. 手动输入各属性值

## 💡 建议

- 使用温度相关的材料属性可以提高精度
- 确保单位系统一致
