# COMSOL 安装与配置

本教程将指导你完成 COMSOL Multiphysics 的安装和基本配置。

## 📋 系统要求

### 硬件要求
| 配置 | 最低要求 | 推荐配置 |
|------|----------|----------|
| **处理器** | Intel/AMD 64位 | 多核处理器 |
| **内存** | 4 GB | 16 GB 或更高 |
| **硬盘** | 10 GB | SSD 50 GB+ |
| **显卡** | 支持 OpenGL | 独立显卡 |

### 操作系统
- Windows 10/11 (64位)
- Linux (Ubuntu, CentOS, RHEL)
- macOS (部分功能受限)

## 🔧 安装步骤

### 步骤 1：下载安装包
1. 访问 [COMSOL 官网](https://www.comsol.com)
2. 登录或注册账号
3. 下载对应系统的安装包

### 步骤 2：运行安装程序
```bash
# Windows
双击运行 comsol_xxx_windows.exe

# Linux
chmod +x comsol_xxx_linux.sh
./comsol_xxx_linux.sh
```

### 步骤 3：选择安装选项
- **典型安装**：包含所有模块（推荐）
- **自定义安装**：选择需要的模块

### 步骤 4：配置许可证
1. **单机许可证**：输入序列号
2. **网络许可证**：配置许可证服务器地址

## ⚙️ 基本配置

### 配置文件路径
```
Windows: C:\Users\<用户名>\AppData\Roaming\COMSOL\
Linux: ~/.comsol/
```

### 推荐设置
1. **内存设置**
   - 编辑 → 首选项 → 内存
   - 设置最大内存使用量

2. **自动保存**
   - 编辑 → 首选项 → 保存
   - 启用自动保存，设置间隔时间

3. **图形设置**
   - 编辑 → 首选项 → 图形
   - 根据显卡调整渲染设置

## 🔍 验证安装

### 测试案例
1. 打开 COMSOL
2. 选择 **文件 → 新建**
3. 选择 **传热模块 → 固体传热**
4. 点击 **完成**
5. 点击 **计算** 按钮
6. 查看结果是否正常

### 常见问题
| 问题 | 解决方案 |
|------|----------|
| 启动失败 | 检查显卡驱动，更新 OpenGL |
| 内存不足 | 增加系统内存或调整设置 |
| 许可证错误 | 检查许可证配置 |
