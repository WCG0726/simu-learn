# COMSOL 安装与配置

本教程将指导你完成 COMSOL Multiphysics 的安装和基本配置。

## 📋 系统要求

### 硬件要求
| 配置 | 最低要求 | 推荐配置 |
|------|----------|----------|
| **处理器** | Intel/AMD 64位 | 多核处理器 (8核+) |
| **内存** | 4 GB | 32 GB 或更高 |
| **硬盘** | 10 GB | SSD 100 GB+ |
| **显卡** | 支持 OpenGL | 独立显卡 (4GB+显存) |

### 操作系统
- Windows 10/11 (64位) ✅ 推荐
- Linux (Ubuntu 20.04+, CentOS 7+, RHEL 7+)
- macOS (部分功能受限)

## 🔧 安装步骤

### 步骤 1：下载安装包
1. 访问 [COMSOL 官网](https://www.comsol.com)
2. 登录或注册账号（需要许可证）
3. 选择版本下载（推荐最新稳定版）

### 步骤 2：运行安装程序

**Windows:**
```
双击运行 comsol_xxx_windows.exe
→ 选择安装语言
→ 接受许可协议
→ 选择安装类型
```

**Linux:**
```bash
chmod +x comsol_xxx_linux.sh
./comsol_xxx_linux.sh
```

### 步骤 3：选择安装选项

| 选项 | 说明 | 推荐 |
|------|------|------|
| **典型安装** | 安装所有模块 | ✅ 初学者推荐 |
| **自定义安装** | 选择需要的模块 | 高级用户 |

### 步骤 4：选择模块

常用模块推荐：
- ☑️ COMSOL Multiphysics（核心）
- ☑️ Heat Transfer Module（传热）
- ☑️ AC/DC Module（电磁）
- ☑️ Structural Mechanics Module（结构）
- ☑️ LiveLink for MATLAB（可选）

### 步骤 5：配置许可证

**单机许可证：**
1. 输入序列号
2. 选择"自动激活"

**网络许可证：**
1. 输入许可证服务器地址：`port@server`
2. 例如：`27000@license-server.company.com`

## ⚙️ 基本配置

### 推荐设置

**1. 内存设置**
```
编辑 → 首选项 → 内存
→ 最大内存使用量：设置为物理内存的 70-80%
→ 例如：32GB 内存设置 24GB
```

**2. 自动保存**
```
编辑 → 首选项 → 保存
→ 启用自动保存：✓
→ 保存间隔：10-15 分钟
→ 保存位置：选择有足够空间的目录
```

**3. 图形设置**
```
编辑 → 首选项 → 图形
→ 渲染器：选择"高性能"
→ 抗锯齿：4x 或更高
```

**4. 临时文件目录**
```
编辑 → 首选项 → 文件
→ 临时文件目录：选择 SSD 磁盘
```

## 🔍 验证安装

### 测试案例 1：简单传热
1. 打开 COMSOL
2. 选择 **文件 → 新建**
3. 选择 **传热模块 → 固体传热 (ht)**
4. 点击 **完成**
5. 创建 2D 矩形（10cm × 10cm）
6. 设置边界条件：
   - 左边：T = 100°C
   - 右边：T = 0°C
7. 点击 **计算** 按钮
8. 查看温度分布图

### 测试案例 2：结构分析
1. 选择 **结构力学模块 → 固体力学 (solid)**
2. 创建 3D 长方体
3. 设置材料为 Steel
4. 固定一端，另一端施加力
5. 计算并查看应力分布

## 🚨 常见问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 启动失败 | 显卡驱动问题 | 更新显卡驱动，安装最新 OpenGL |
| 内存不足 | 物理内存不够 | 增加内存或减小模型规模 |
| 许可证错误 | 许可证配置错误 | 检查许可证服务器连接 |
| 计算很慢 | 网格太细或模型太大 | 使用更粗的网格或并行计算 |
| 界面显示异常 | DPI 缩放问题 | 调整 Windows 显示缩放为 100% |
